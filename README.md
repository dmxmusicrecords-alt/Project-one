# ⚛️ Moscovium115 — Escrow-Secure Global Marketplace

**Moscovium115** is a state-of-the-art secure global marketplace for trading physical products, on-demand custom services, and digital downloads. It provides a flawless client experience, rich responsive slideshows, localized multi-language controls (English & Kiswahili), escrow transactions tracking, and a built-in merchant portal.

---

## 🚀 Key App Components & Features

1. **Marketplace Center**:
   - Filter items dynamically by clicking top department categories (Electronics, Collectibles, Services, etc.).
   - Search the entire catalog using the live, fluid header search bar.
   - Beautiful visual fallbacks for items without images using CSS gradient themes.

2. **Secure Escrow Checkout**:
   - Dynamic Shopping Cart tracking item quantities.
   - Interactive transactional simulation syncing balances with mock payment platforms.
   - Moves fulfilled orders immediately into user purchase history.

3. **Merchant Office & Wallets**:
   - Interactive seller portal tracking lifetime earnings and current available capital.
   - Real-time withdraw system: requests on-demand cashouts (PayPal, Wise, SWIFT wires) that subtract from local balances dynamically.
   - Live Listing Form: add custom crafts, books, or consultancies and see them appear instantly in the marketplace!

4. **Git Deploy Assistant**:
   - Tailor customized, copy-and-paste command structures.
   - Step-by-step guidance on initializing Git, identity configurations, staging code, linking origin remotes, and pushing securely.

5. **Integrated Live Chat Helper**:
   - Interactive, context-aware AI support chat simulator on the bottom right of the screen for absolute transactional confidence.

## 🛠️ Step-by-Step Git Upload Instructions

Moscovium115 has a built-in **Git Deploy Assistant** right inside the app! However, here is the universal command sequence to push this code onto your GitHub or GitLab repository:

```bash
# 1. Initialize local repository
git init

# 2. Add files to staging area
git add .

# 3. Create your first savepoint
git commit -m "Initial launch: Moscovium115 Marketplace Beta v1.0"

# 4. Configure local main branch
git branch -M main

# 5. Link origin to your newly created online repository
git remote add origin https://github.com/your-username/moscovium115.git

# 6. Push code (requires a personal access token or SSH authorization)
git push -u origin main
```

# 3. Compile for production
